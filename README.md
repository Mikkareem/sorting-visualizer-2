# Sorting Visualizer  (ReactJS Project)

This project is for visualizing the Major Sorting algorithms.
1) Merge Sort
2) Quick Sort
3) Selection Sort
4) Insertion Sort
5) Heap Sort
6) Bubble Sort


The Visualization tool is available here [Github Page](https://mikkareem.github.io/sorting-visualizer-2)
